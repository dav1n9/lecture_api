# lecture_api

## 주요 기능

- spring security 를 사용하여 인증/인가 구현
- access token 과 refresh token 활용한 로그인
- 강의 좋아요 및 무한 댓글 작성 기능
- AWS EC2 와 RDS 사용하여 프로젝트 배포
- Github Actions 를 이용하여 CI/CD  => 진행중
